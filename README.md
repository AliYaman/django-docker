## Django Development With Docker Compose

Requirements:

- Docker v1.10.3+
- Docker Compose v1.6.2+
- Docker Machine v0.6.0+
- Python 3.5+

Used technologies :

- Nginx
- Gunicorn
- Postgresql


### Installation 

    sh install.sh
